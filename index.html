Frequently Asked Questions

Pods for Chrome is a modern multi-login extension for your browser. It allows you to manage multiple accounts seamlessly. It also has effortless integration with reverse proxies.

Pods for Chrome was created in response to the high demand from users for a fast and efficient multi-session solution for Chromium-based browsers. Unlike existing solutions, our graphical interface is not so complicated that you almost need a course to learn how to use it.

Currently the extension does not collect any user information. The permissions are explicitly described in the Chrome WebStore and are used exclusively for operation, that is, all user interaction information is stored locally in your browser, and never leaves your machine.
